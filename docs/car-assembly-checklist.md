# 2WD Car Assembly Checklist

- [ ] Assemble chassis and mount motors/wheels
- [ ] Connect motor driver to motors and battery
- [ ] Connect UNO control pins to motor driver inputs
- [ ] Verify common GND across all modules
- [ ] Run motor forward/backward test per wheel
- [ ] Mount ultrasonic and infrared sensors securely
- [ ] Test basic movement + sensor reads together
- [ ] Start integration sketch for navigation logic
